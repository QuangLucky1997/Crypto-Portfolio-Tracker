🚀 Crypto Portfolio Tracker
A modern Android application to track, manage, and analyze your cryptocurrency portfolio in real time. Built with Kotlin and clean architecture to deliver fast, secure, and user-friendly experiences for crypto investors.
✨ Features
📊 Track crypto assets in real time
🔍 Search and add coins to wathlist
🤖 Chatbot predicts and analyzes tokens.
🌙 Dark mode support
📡 Live market price updates

🛠 Tech Stack
Language: Kotlin
Architecture: MVVM
UI: XML
Async: Kotlin Coroutines & Flow
Network: Retrofit
Dependency Injection: Hilt
Database: Room
API: Crypto Market API (Binance / CoinGecko, CoinMarketCap)
Admod , Remote Firebase Config , Crashlytic, Analytic
